# 测试系统管理平台

一个功能完整的测试系统Web应用，提供测试用例管理和测试任务管理功能。

## 功能特性

### 测试用例管理
- ✅ 创建、编辑、删除测试用例
- ✅ 测试用例信息包括：名称、描述、优先级、状态、测试步骤、预期结果
- ✅ 支持按状态、优先级筛选
- ✅ 支持搜索功能
- ✅ 状态管理：待执行、执行中、已通过、已失败

### 测试任务管理
- ✅ 创建、编辑、删除测试任务
- ✅ 测试任务信息包括：名称、描述、关联测试用例、执行人、截止日期、状态
- ✅ 支持按状态筛选
- ✅ 支持搜索功能
- ✅ 状态管理：待执行、执行中、已完成、已取消

## 技术栈

- **后端**: Flask (Python)
- **前端**: HTML5 + CSS3 + JavaScript (原生)
- **数据存储**: JSON文件
- **API**: RESTful API

## 安装和运行

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行应用

```bash
python app.py
```

或者使用 Python 3：

```bash
python3 app.py
```

### 3. 访问应用

打开浏览器访问：`http://localhost:5000`

## API接口

### 测试用例API

- `GET /api/test-cases` - 获取所有测试用例
- `POST /api/test-cases` - 创建新测试用例
- `PUT /api/test-cases/<id>` - 更新测试用例
- `DELETE /api/test-cases/<id>` - 删除测试用例

### 测试任务API

- `GET /api/test-tasks` - 获取所有测试任务
- `POST /api/test-tasks` - 创建新测试任务
- `PUT /api/test-tasks/<id>` - 更新测试任务
- `DELETE /api/test-tasks/<id>` - 删除测试任务

## 使用说明

### 测试用例管理

1. 点击"测试用例管理"标签页
2. 点击"+ 新建测试用例"按钮创建新用例
3. 填写用例信息：
   - 用例名称（必填）
   - 描述
   - 优先级（高/中/低）
   - 状态
   - 测试步骤（每行一个步骤）
   - 预期结果
4. 使用搜索框和筛选器快速查找用例
5. 点击"编辑"或"删除"按钮管理用例

### 测试任务管理

1. 点击"测试任务管理"标签页
2. 点击"+ 新建测试任务"按钮创建新任务
3. 填写任务信息：
   - 任务名称（必填）
   - 描述
   - 执行人
   - 截止日期
   - 状态
   - 关联测试用例ID（用逗号分隔，如：1,2,3）
4. 使用搜索框和筛选器快速查找任务
5. 点击"编辑"或"删除"按钮管理任务
